Sobre

Aplicativo desktop para controle de vendas e estoque de carvão.
Distribuição self-contained (não precisa instalar .NET no computador do cliente).

Requisitos

Windows 10 ou 11 64 bits

Permissão de escrita no perfil do usuário (AppData)

Instalação / Execução

Baixe o arquivo .zip disponível na pasta release.

Extraia para qualquer pasta (ex.: C:\CarvaoControl).

Execute CarvaoControl.exe.

Primeiro acesso (Administrador)

Senha padrão: chamacarvao

Para alterar: Menu Administrador → Alterar Senha

Funcionalidades principais
Vendas

Registrar nova venda

Listar vendas

Relatórios

Estoque

Controle de estoque

Cadastrar produto

Renomear produto

Ajustar quantidade (exige admin)

Limpar produtos (ação irreversível)

Backup

Menu Administrador → Backup

Gera um .zip com todos os dados

Usuário escolhe onde salvar

Versão Single-File (release)

A pasta release contém:

CarvaoControl.exe

Executável único

Empacotado como self-contained + single-file

Não exige instalação do .NET

Como executar

Baixe o ZIP

Extraia

Clique em CarvaoControl.exe

Observações

Pode existir uma pasta Resources/ para ícones e imagens

Na primeira execução, o modo single-file pode extrair arquivos temporários (comportamento normal)
